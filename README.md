# Gym Point !! The Fitness Studio 🏋️‍♂️

This is a Gym Management System project built using Python and Django.

 Feature
- Add and manage gym members
- Track member registration and fees
- Clean frontend using HTML, CSS, and Bootstrap
- Django backend with SQLite database

## 🛠️ Tech Stack

- **Backend:** Python, Django
- **Frontend:** HTML, CSS, Bootstrap
- **Database:** SQLite
